## Try it out in your terminal:
Clone the project
```bash
git clone https://github.com/MaxKulakov/which.toad.am.i.git
```

Go to directory
```bash
cd which.toad.am.i
```

Run 
```bash
python toad.py
```

Or just copy command
```bash
git clone https://github.com/MaxKulakov/which.toad.am.i.git && cd which.toad.am.i && python toad.py
```
