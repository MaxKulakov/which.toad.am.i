# which.toad.am.i

The project was created to find out what kind of toad you are

## Try it out in your terminal:
Clone the project
```bash
git clone https://github.com/MaxKulakov/which.toad.am.i.git
```

Go to directory
```bash
cd which.toad.am.i
```

Run 
```bash
python toad.py
```
